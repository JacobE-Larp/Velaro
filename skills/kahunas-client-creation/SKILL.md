# Kahunas Client Creation

## Purpose

Create a new client inside the Kahunas platform by following a strict, step-by-step onboarding workflow.

## When to Use

Use this skill when:
- A coach asks to create a new client
- A client needs to be onboarded into Kahunas
- Client setup and configuration is required

## Required Inputs

Before starting, confirm all of the following are available:
- First name
- Last name
- Email
- Date of birth
- Phone number
- Payment plan
- Payment method (Kahunas or external)
- Forms to assign (Q&A, daily habit, check-in)

Do NOT proceed if any required information is missing.

## Workflow

1. Open Kahunas dashboard
2. Click "Add Client"
3. Select "Add Single Client"
4. Enter all required client details
5. Select payment plan and method
6. Set plan start date
7. Set measurement unit (default: kg)
8. Assign:
   - Q&A form
   - Daily habit form
   - Check-in form
9. Set:
   - Check-in frequency: weekly
   - Check-in day: Sunday
10. Toggle ON:
    - Goal setting
    - Access to library videos
    - Apple Pay
11. Review all fields
12. Only then click "Add Client"

## Constraints

- Never skip steps
- Never guess missing info
- Never leave fields blank
- Do not submit until everything is complete
- Confirm forms before assigning

## Output Behaviour

- Confirms all inputs before starting
- Walks through process step-by-step
- Flags missing info immediately
- Confirms completion once client is created
