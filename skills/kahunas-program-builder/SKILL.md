# Kahunas Training Program Builder

## Purpose

Design and upload a structured training program inside Kahunas based on client goals, experience, and availability.

## When to Use

Use this skill when:
- A coach asks to create a program
- A client needs a training plan
- Program design or upload is required

## Required Inputs

Before starting, confirm all of the following are available:
- Client Q&A form
- Training frequency (days per week)
- Client goals
- Injury history or limitations

Do NOT proceed if this information is missing.

## Workflow

1. Review client Q&A form
2. Determine training split based on:
   - Days available
   - Goals
3. Open Kahunas → Workout → Workout Programs
4. Click "Add Workout Program"
5. Enter:
   - Program title (Client name + split)
   - Client name
   - Description
6. Build weekly structure:
   - Add training days
   - Add rest days
7. For each training day:
   - Add warm-up exercises
   - Add main exercises
8. For each exercise, specify:
   - Sets (minimum required)
   - Reps / rep range
   - Rest time
   - RPE
   - Tempo
9. Verify:
   - All muscle groups covered
   - Volume within guidelines
10. Review full program
11. Present to head coach for approval
12. Only upload after approval is received

## Constraints

- Never upload without approval
- Never use generic templates blindly
- Always personalise the program
- Always include warm-ups
- Never program below minimum sets
- Always verify unclear data before proceeding

## Output Behaviour

- Builds structured program step-by-step
- Explains reasoning when needed
- Requests approval before upload
- Confirms once program is complete and uploaded
