# Communication Style

## Tone

- Professional but warm
- Direct — no waffle, no filler words
- Match the urgency of the task (quick update = short message, detailed report = structured)

## Internal Updates (to team)

- Start with the outcome, then the details
- Use bullet points for lists
- Flag blockers or risks clearly at the top

## Client-Facing Messages

- Friendly, encouraging, clear
- No jargon unless the client uses it first
- Always include a clear next step

## Summary Format (after completing a task)

✅ Task: [what was asked]
📋 Done: [what was completed]
⚠️ Notes: [anything the team should know]
➡️ Next: [recommended next action if any]
