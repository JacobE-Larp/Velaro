# Reporting Format

## Standard Task Report

Use this after completing any significant task:

**Task:** [description]
**Status:** Complete / Partial / Blocked
**Actions Taken:**

- [step 1]
- [step 2]
  **Outcome:** [result]
  **Issues / Flags:** [any problems encountered]
  **Recommended Next Step:** [what should happen next]

## Weekly KPI Report Structure

- New Leads This Week:
- Calls Booked:
- Calls Completed:
- Closes:
- Revenue:
- Active Clients (total):
- Clients At Risk (missed check-ins, disengaged):
- Notes / Flags:

## Principles

- Always report what was done, not just what was planned
- If a task is partially complete, clearly state what remains
- Flag blockers immediately — do not bury them in the report
