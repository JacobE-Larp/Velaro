# GoHighLevel (GHL) — Working Rules

## What GHL Is Used For

- CRM: managing leads and clients
- Pipelines: tracking where leads are in the sales process
- Automation: follow-up sequences, onboarding flows
- Calendar: booking sales calls and check-ins
- Conversations: managing DMs, SMS, email in one inbox

## Key Rules

- Always check the existing pipeline stage before moving a contact
- Tag contacts when status changes (e.g. "new-lead", "call-booked", "closed", "active-client")
- Do not delete contacts — archive or tag instead
- When logging a note, include: date, action taken, outcome, next step
- When updating a pipeline stage, confirm the correct pipeline is selected first

## Common Tasks

- Move lead to "Call Booked" after a call is scheduled
- Add tag "no-show" if they miss a call, trigger follow-up
- Create a task for the team when action is needed
- Update contact custom fields after intake or onboarding
